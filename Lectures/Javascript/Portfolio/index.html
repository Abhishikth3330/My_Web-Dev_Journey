  • Implemented image preprocessing steps like resizing, grayscale conversion, normalization, and augmentation to improve model performance.
  • Used TensorFlow and Keras for model development, OpenCV for image handling, and Matplotlib for performance visualization.
  • Designed the pipeline to classify images as “cracked” or “non-cracked,” supporting automated inspection and structural health monitoring.
  • The model demonstrated robustness on unseen test data, supporting integration into IoT-based monitoring systems for real-time analysis.
